# Kelvin, AI Workflow Automation Platform

Kelvin is an AI-powered workflow automation tool that helps developers interact with GitHub and Notion using natural language. It connects real developer tools with LLM-driven actions to reduce repetitive work and speed up everyday tasks.

This project is actively under development and serves as a portfolio project demonstrating real-world full-stack engineering skills.

## What Kelvin Does

Kelvin allows users to log in, connect their tools, and perform actions using AI-driven workflows.

### Authentication
- Google OAuth
- Email-based authentication

### GitHub Integration
Users can:
- Create GitHub issues using AI
- Summarise pull requests
- Edit existing files and create a pull request
- Create an entirely new pull request using AI-generated changes

### Notion Integration
Users can:
- Summarise existing Notion pages using AI
- Create new Notion page entries generated by AI

### Extras
- View past prompts and generated summaries
- Persistent history per user

## Why This Project Exists

The goal of this project was to:
- Build a real AI-powered product, not a chatbot clone
- Work with real third-party APIs (GitHub, Notion)
- Handle authentication, permissions, and side effects safely
- Design an agent-style workflow instead of prompt dumping

This project focuses on **integration, orchestration, and deployment**, not just AI responses.
